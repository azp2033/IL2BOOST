Компиляция

Ну для сборки исходного кода, необходимо его сначала скачать, так-же потребуется Visual Studio 2019 или более свежая с поддержкой классических приложений C++, свежим SDK а так-же компилятором версии 142 или более новым.

После загрузки исходника распаковываем его и открываем в Visual Studio

  1. Первый шаг
    Скачиваем и распаковываем исходник
  2. Далее
    Запускаем исходник в Visual Studio
  3. После этого
    Начинаем редактировать файл Config.hpp, спускаемся вниз и видим такой кусок кода
    static std::vector Binds = { IBind(u8"Принять ТП", "chat.say /tpa", 'Y'), IBind(u8"Отменить ТП", "chat.say /tpc", 'N'), IBind(u8"Киты", "chat.say /kit", 'K'), IBind(u8"Меню сервера", "chat.say /menu", 'M'), }; static const char* Title = u8"IL2BOOST by azp2033"; static int MenuKey = VK_INSERT; namespace Colors { static Color Primary(68, 68, 255); static Color Secondary(51, 51, 57); static Color Text(255, 255, 255); static Color Background(34, 34, 41); };
    тут мы можем настроить бинды, изменить заголовок, кнопку открытия меню, а так-же заменить цвета бустера.
  4. Следующим этапом
    Мы запускаем сборку кода и на выходе получаем два файла - Запускатор и сам бустер

Для запуска бустера запустите игру, после чего запустите лаунчер бустера..

Обновление оффсетов

Для обновления оффсетов нам необходимо загрузить IL2CPPDumper, благодаря нему мы их и сможем получить..

  1. Первый шаг
    Скачиваем и распаковываем IL2CPPDumper
  2. Следующим этапом
    Переходим на рабочий стол после чего нажимаем сочитание клавиш + R, в открывшееся окно мы пишем cmd.exe
  3. После этого
    После открытия консоли, мы копируем путь до папки с распакованным IL2CPPDumper'ом и пишем команду в консоль cd путь_до_папки
  4. Следующая команда
    Il2CppDumper.exe путь_до_папки_с_игрой/GameAssembly.dll путь_до_папки_с_игрой/RustClient_Data/il2cpp_data/Metadata/global-metadata.dat
  5. По итогу выполнения команды
    Мы видим что в папочке куда вы распаковали IL2CPPDumper появился файл Dump.cs, он их хранит наши оффсеты..
  6. Сами оффсеты
    Нам необходимо 5 штук
    1. В классе SunSettings берем RVA от метода Update() в моем случае это 0x71FBE0 его мы ставим в SunSettings::Update
    2. В классе SunSettings берем оффсет от поля light в моем случае это 0x18 его мы ставим в Light::LightInstance
    3. В классе FoliageCell берем RVA от метода CalculateLOD() в моем случае это 0x7ED120 его мы ставим в FoliageCell::CalculateLOD
    4. В классе Light берем RVA от метода set_shadows(LightShadows value) в моем случае это 0x1894C20 его мы ставим в Light::SetShadowsQuality
    5. В классе ConsoleSystem берем RVA от метода SendToServer(string command) в моем случае это 0x21ED8F0 его мы ставим в ConsoleSystem::SendToServer
  7. Готово
    Компилируем исходник и используем :)
